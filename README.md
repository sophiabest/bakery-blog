## Getting Started

<details>
<summary>Walkthrough</summary>
 
1. First, the user should log in.

2. Next, the user can now add posts.
 
3. Lastly, the user can comment on other posts or their own.
</details>

<details>
<summary>Walkthrough CRUD</summary>
 
1. The user can create a post by clicking new post.
 
2. The user can delete posts or comments by clicking delete.
 
3. The user can edit and update a comment in line with the page. 

 
</details>
